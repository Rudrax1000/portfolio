# Portfolio

This project is built using React, TypeScript, and Vite.

## Tech Stack
- React
- TypeScript
- Vite
- ESLint

## About the Project
This portfolio showcases my work, projects, and skills in UI/UX and web development.

## Development Setup

This template provides a minimal setup to get React working in Vite with HMR and ESLint rules.

### Available Plugins
- @vitejs/plugin-react (uses Oxc)
- @vitejs/plugin-react-swc (uses SWC)

## Notes

- ESLint can be expanded for stricter and type-aware rules.
